# _data.nosync/derived

Optional derived-memory engine artifacts live here.

Generate corpora with:

```bash
oracle derived-memory prepare mempalace
oracle derived-memory prepare graphify
```

Each export is capped by sensitivity and generated from Oracle's rebuildable
`knowledge_index`. External tools may index or analyze these files, but the
outputs remain derived. Promote useful discoveries back through review-gated
Findings, Questions, Contradictions, Sources, or truth-map updates before they
influence material answers.

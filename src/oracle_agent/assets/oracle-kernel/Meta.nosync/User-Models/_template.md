## Who

Name / role / mandate. What is this person accountable for?

## Decision Style

How do they decide? (fast vs deliberate, data-first vs intuition-first, risk
posture, who else they answer to)

## What They Value

What makes an answer useful to *them* specifically — speed, certainty,
optionality, defensibility, brevity, depth.

## Output Preferences

- Ranges vs point estimates.
- Prose vs tables vs dashboards.
- Level of hedging they tolerate.
- How much supporting evidence they want surfaced.

## Distrust Triggers

What makes this user stop trusting the oracle? (e.g. confident wrong answers,
over-hedging, burying the lede, stale data presented as fresh)

## Standing Questions

The recurring things this user asks the oracle.
